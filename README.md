# AI Video Studio 🎬

Free AI video generator tool using multi-API system.

## Features
- Runway-style UI
- Multi API (ZSky + fal.ai)
- Offline fallback
- Blogger compatible frontend

## Setup

### Backend
cd backend
npm install
cp .env.example .env
npm start

### Frontend
Open index.html
Replace:
https://YOUR-BACKEND-URL

## Deploy
- Backend → Render / Railway
- Frontend → Blogger / GitHub Pages
